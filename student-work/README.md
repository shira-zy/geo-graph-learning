# student-work/ (root)

Optional cross-unit workspace. Use this if you want a single place for
experiments, notes, or projects that span more than one unit.

Per-unit `student-work/` directories (under each `unit-N-*/` folder) are
the primary place to put your in-class and homework deliverables. This
root-level directory is for everything else.

**This directory is "yours."** The upstream repo only ships this README
and a `.gitkeep`. The instructor's `promoter` agent NEVER touches files
here, so your work won't conflict when you sync upstream.
