# Direct → Interpret → Extend Rubric

The five checks you apply every supervised-practice hour. Memorize by Unit 3.

## DIRECTING

- [ ] Did I phrase my request using **this unit's vocabulary**
      (e.g., "compute betweenness centrality" — not "find important streets")?
- [ ] Did I specify the inputs precisely enough that the agent
      didn't have to guess (which city, which metric, which time window)?

## INTERPRETING

- [ ] Can I explain the result back **in my own words**, in domain terms?
- [ ] Did I notice anything that didn't match my expectation?
      (a sign of either a bug or a real insight — both worth chasing)

## EXTENDING

- [ ] Did I ask a **follow-up question** grounded in what the result revealed?
